# ai-rule-based-chatbot
This project is rule-based chatbot developed using python as a part of an internship at Sam IT Technologies. it interacts with users through predefined rules and keyword-based responses
# Features
-Greetings Responses
-Basic Q&A using keywords
-Simple terminal interaction
# how to run
1.Open VS Code
2.Run
## 📌 Output

After running the program, the chatbot interacts with the user in the terminal based on input keywords.

### Example:
You: hello  
Bot: Hello! 😊 How can I help you today?

You: what is python  
Bot: Python is a programming language used in AI, ML, and web development.
